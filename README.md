The **Raeen Dashboard** is a high-performance productivity workspace engineered for total immersion and data sovereignty. Transform your "New Tab" page into a professional command center featuring a Focus Timer, Quick Notes, and customizable aesthetics.

---

## ✨ Key Features

- 🕒 **Focus Timer**: Integrated Pomodoro-style timer to keep you in the zone.
- 📝 **Quick Notes**: Instant scratchpad that saves locally and persists across sessions.
- 🖼️ **Immersive Backgrounds**: Beautiful, high-resolution visuals that adapt to your style.
- 🔒 **Privacy First**: Zero data collection. Everything stays on your local hardware.
- ⚡ **Lightweight & Fast**: Built with vanilla JS for near-instant load times.
- 🔍 **Secure Search**: Direct API bridges for suggestions without tracking.

## 🎨 Design Philosophy

Raeen Dashboard utilizes an **Ethereal Glass / Double-Bezel** architecture, combining the clarity of modern glassmorphism with the structural integrity of professional UI design.

- **Typography**: Optimized with *Plus Jakarta Sans* & *Playfair Display*.
- **Aesthetics**: Sleek dark mode by default with vibrant micro-animations.

## 🛠️ Installation

### For Developers (Manual Install)

1. **Clone the repository**:
   ```bash
   git clone https://github.com/abdussamadraeen/Raeen-Dashboard.git
   ```
2. **Open Chrome/Edge**: Navigate to `chrome://extensions` or `edge://extensions`.
3. **Enable Developer Mode**: Toggle the switch in the top right corner.
4. **Load Unpacked**: Click "Load unpacked" and select the `NewTab_Lite` folder.

## 🛡️ Privacy & Security

We believe in **Local Sovereignty**. 
- **Zero Data Collection**: No personal data is ever collected or transmitted.
- **Local Storage**: Your notes, timers, and settings are stored exclusively in your browser's local storage.
- **Permissions**: We only request necessary permissions for core functionality.
