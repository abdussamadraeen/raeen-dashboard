# Raeen Dashboard

Official landing page and privacy protocol for the **Raeen Dashboard** browser extension.

## Overview
Raeen Dashboard is a high-performance productivity workspace engineered for total immersion and data sovereignty. This repository hosts the official landing page and the technical privacy specifications required for the Chrome Web Store and Microsoft Edge Add-ons.

## Privacy & Security
We take a privacy-first approach to productivity:
- **Zero Data Collection**: No personal data is ever collected or transmitted.
- **Local Sovereignty**: All user data (notes, timers, settings) is stored exclusively on the user's local hardware.
- **Secure Bridging**: Search suggestions utilize direct, secure API bridges to global providers.

## Technical Stack
- **Design Architecture**: Ethereal Glass / Double-Bezel
- **Typography**: Plus Jakarta Sans & Playfair Display
- **Deployment**: GitHub Pages
